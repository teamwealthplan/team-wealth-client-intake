# Team Wealth Plan — Client Intake Form (Financial OPD)

ระบบกรอกข้อมูลการเงินเบื้องต้นสำหรับลูกค้าเพื่อส่งข้อมูลการเงินให้นักวางแผนการเงินวิเคราะห์ต่อ

---

## 🎯 วัตถุประสงค์
เครื่องมือนี้จัดทำขึ้นเพื่อให้ลูกค้าของ **Team Wealth Plan** สามารถกรอกข้อมูลประวัติการเงินส่วนตัว รายได้ ค่าใช้จ่าย สินทรัพย์ หนี้สิน และเป้าหมายทางการเงินต่าง ๆ ได้อย่างสะดวกและปลอดภัยผ่านระบบ Web Interface บนเครื่องของตนเอง โดยข้อมูลทั้งหมดจะไม่ถูกอัปโหลดขึ้นเซิร์ฟเวอร์หรือคลาวด์ใด ๆ เพื่อความเป็นส่วนตัวสูงสุดของข้อมูลการเงินลูกค้า

เมื่อกรอกเสร็จสิ้น ลูกค้าสามารถส่งออกไฟล์ข้อมูลการเงิน (.json) เพื่อส่งกลับมาให้ที่ปรึกษาการเงิน/นักวางแผนการเงินนำไปวิเคราะห์ในระบบ **Planner Workspace** ต่อไป

---

## 🛠️ วิธีการใช้งานสำหรับลูกค้า
1. **เปิดใช้งาน**: ดับเบิลคลิกเปิดไฟล์ `index.html` ในโฟลเดอร์นี้ด้วยเว็บเบราว์เซอร์ใดก็ได้ (แนะนำ Chrome, Safari, หรือ Edge)
2. **กรอกข้อมูล**: กรอกข้อมูลการเงินตามระบบแนะนำแบบเป็นขั้นตอน (Wizard Flow):
   - ข้อมูลส่วนตัวและครอบครัว (Profile)
   - รายได้ประเภทต่าง ๆ (Income)
   - ค่าใช้จ่ายประเภทต่าง ๆ (Expenses)
   - สินทรัพย์สภาพคล่อง สินทรัพย์ลงทุน และสินทรัพย์ส่วนตัว (Assets)
   - หนี้สินภาระผูกพันต่าง ๆ (Debts)
   - เป้าหมายทางการเงินหลัก (Goals)
3. **ตรวจสอบและบันทึกผล**: เมื่อกรอกครบถ้วน ระบบจะสรุปคะแนนสุขภาพการเงินเบื้องต้น (Financial Health Score) และตารางสรุปกระแสเงินสดและงบดุลให้ตรวจสอบ
4. **ส่งออกข้อมูล**:
   - กดปุ่ม **"บันทึกไฟล์ข้อมูลของฉัน"** เพื่อดาวน์โหลดไฟล์ `.json` (เช่น `TWP_Client_[ชื่อ]_[ปี]_FullData.json`)
   - กดปุ่ม **"ส่งออก PDF"** เพื่อพิมพ์รายงานสรุปสุขภาพการเงินเก็บไว้
5. **ส่งกลับให้นักวางแผน**: ส่งไฟล์ข้อมูล `.json` ที่ดาวน์โหลดไว้กลับไปให้นักวางแผนการเงินผ่านช่องทางที่ติดต่อกัน

---

## 🔒 นโยบายความเป็นส่วนตัวของข้อมูล (Data Privacy)
- **Local-First & Offline**: ระบบนี้เป็นระบบทำงานฝั่งเบราว์เซอร์ลูกค้าทั้งหมด 100% ไม่ต้องการการเชื่อมต่ออินเทอร์เน็ตและไม่มีการเก็บข้อมูลลงในเซิร์ฟเวอร์ใด ๆ ของบริษัท
- **Auto-Save**: ระบบจะบันทึกข้อมูลที่กรอกไว้โดยอัตโนมัติในเบราว์เซอร์ของลูกค้า (ผ่าน `localStorage` คีย์ `twp_opd_client_self`) หากเผลอปิดหน้าต่างไป สามารถกลับมาเปิดใหม่เพื่อกรอกต่อได้ทันที
- **Data Erasure**: ลูกค้าสามารถกดปุ่ม **"ล้างข้อมูลปีนี้"** เพื่อลบข้อมูลที่กรอกไว้ในรอบปีนั้นออกจากเบราว์เซอร์ได้ทันที
